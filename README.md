##  Estructura del Repositorio

El laboratorio está segmentado en entornos lógicos para separar la investigación teórica de la explotación práctica:

*    **`/Machines`**: Writeups detallados y scripts de explotación para máquinas completadas (ej. *Kobold - Hack The Box*).
*    **`/Labs`**: Entornos de prueba aislados y prácticas de enumeración (Footprinting).
*    **`/Tools`**: Scripts propios, automatizaciones en Python/Bash y utilidades de reconocimiento (ej. *ReconSpider*, *FinalRecon*).
*    **`/academy` & `/ctfs`**: Notas estructuradas de módulos de HTB Academy y resoluciones de competiciones CTF.
*    **`/Notes` & `/Hacking`**: Bitácora de guerra, metodologías teóricas y documentación histórica.
*    **`/Outputs`**: Directorio de volcado para *hashes*, resultados de *Nmap* (`.json`) y artefactos temporales (Ignorado parcialmente para mantener limpio el historial).
